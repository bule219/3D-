# Vue
Vue是一个渐进式的JavaScript框架，
## Vue脚手架
1.安装脚手架
cnpm installn -g @vue/cli
2.创建项目
vue vi  图形化创建项目
3.运行项目
cd Vue项目\demo
cnpm run serve
## Vue的目录结构
src:源码文件
babel.config.js:babel的配置文件
